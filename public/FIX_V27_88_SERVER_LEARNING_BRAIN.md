# V27.88 - Server Learning Brain

Miglioramento memoria globale server:

- memoria prodotto aggregata con chiave canonica nome + marca;
- varianti/formati gestiti come voti invece che sovrascritture casuali;
- conteggio conferme e famiglie/utenti diversi;
- affidabilità bassa/media/alta basata su conferme e utenti distinti;
- evidence tokens da nome, marca, etichetta, testo letto, ingredienti e prove visive;
- match memoria più prudente: serve marca/nome coerente oppure più token specifici con categoria compatibile;
- la memoria server non deve comandare da sola se la nuova etichetta dice altro;
- OpenAI resta docente solo per prodotti nuovi/incerti, mentre prodotti confermati più volte possono essere riconosciuti da memoria server.

Obiettivo: apprendimento globale vero, meno contaminazione tra prodotti simili e migliore riuso delle conferme degli utenti.
