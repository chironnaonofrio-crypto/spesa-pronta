# V27.40 - Self Learning Core

Questa versione trasforma la Vision in un sistema teacher-student:

- OpenAI diventa docente temporaneo, non dipendenza fissa.
- Ogni prodotto confermato genera una firma visiva locale.
- Le firme vengono aggregate in modelli locali per prodotto.
- Dopo abbastanza esempi e sicurezza alta, la scansione salta OpenAI e usa l'autonomia locale.
- La memoria salva correzioni su nome, marca, formato, categoria e unità.
- Il core contiene regole native per acqua/bevande, bottiglie, latte, pasta, casa e pet.

Obiettivo realistico: diventare più precisa di OpenAI sui prodotti reali dell'utente, perché impara il suo frigo, la sua dispensa, le sue marche e i formati confermati.
